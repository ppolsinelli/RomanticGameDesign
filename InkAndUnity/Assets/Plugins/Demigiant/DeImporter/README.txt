// GET STARTED /////////////////////////////////////////////////

After importing DeImporter, you can access it via Unity's main menu (Tools > Demigiant > DeImporter). It will open with a Help Panel explaining you how it works. You can re-access the Help Panel at any time via the top-right "?" button.

// IMPORTANT ///////////////////////////////////////////////////

If you already have Demigiant's Core DemiLib libraries in your project, replace them with the ones coming with DeImporter.
